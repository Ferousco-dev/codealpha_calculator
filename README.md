## Overview

A fully functional calculator with a frosted-glass UI and a smooth gradient backdrop. Built with plain HTML, CSS, and JavaScript — no frameworks, no dependencies.

## Features

- ➕ Standard arithmetic operations (add, subtract, multiply, divide)
- 💎 Glassmorphism design with a layered gradient background
- ⌨️ Click-to-input keypad
- 📱 Responsive and lightweight (single file)

## Tech Stack

- **HTML5** · **CSS3** · **Vanilla JavaScript**

## Getting Started

```bash
git clone https://github.com/Ferousco-dev/codealpha_calculator.git
cd codealpha_calculator
# open index.html in your browser
```
